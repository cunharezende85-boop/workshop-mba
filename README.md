# Workshop MBA — IA Aplicada ao Direito

Portal de workshops do MBA em IA Aplicada ao Direito, com a agenda e o material de apoio de cada encontro.

## Acesse o portal

https://institutonta.github.io/workshop-mba/

## Estrutura deste repositório

- `index.html` — página do portal
- `workshops.json` — lista dos workshops (título, data, descrição), exibida automaticamente na página

## Como atualizar os workshops

Para adicionar ou editar um workshop, edite o arquivo `workshops.json` e salve — a página é atualizada automaticamente a partir desses dados.
